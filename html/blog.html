Het Begin van Mijn Zoektocht

Toen ik besloot om een blog te starten, begon ik mijn zoektocht op het internet en overwoog ik eerst om Wix te gebruiken. Ik had Wix al eens eerder gebruikt voor een ander project en wist dat ik maar één gratis website kon maken. Daarom moest ik op zoek naar een andere oplossing.

Experimenteren met Raspberry Pi

Ik overwoog om mijn blog zelf te hosten op mijn Raspberry Pi. Hoewel dit me erg interessant leek, vond ik het instellen van poortforwarding te ingewikkeld en omslachtig. Hierdoor ben ik verder gaan zoeken naar een eenvoudiger alternatief.

GitHub als Mogelijke Oplossing

Mijn volgende idee was om de blog eerst op GitHub te hosten en vervolgens een provider te vinden die GitHub-repositories kan hosten. Maar terwijl ik me hierin verdiepte, vertelde een vriend me dat ik mijn blog ook rechtstreeks via GitHub Pages kon hosten.

De Perfecte Oplossing: GitHub Pages

Dit bleek de perfecte oplossing te zijn. Ik volgde enkele handleidingen en zette mijn blog op via GitHub Pages. Het was een leerzaam proces en nu staat mijn blog online, helemaal gratis en zonder gedoe met poortforwarding of externe providers.

Het opzetten van mijn blog via GitHub Pages was een leerzame ervaring en ik ben blij met het resultaat. Mijn blog staat nu online, en ik kan mijn ervaringen en inzichten gemakkelijk delen met de wereld.
